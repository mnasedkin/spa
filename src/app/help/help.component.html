Business idea: improve look and feel of our web-portals by using modern technologies.
Add effects, animations, etc.

Technologies set:

  • Angular SPA (single page application)
  • Parallax
  • Lazy load for images
  • CSS/JS animation
  • PWA


2. Why Angular?

  • In Top 3 Front-end Frameworks
  • Powered by Google
  • Best Fit for SPA (Single Page Applications)
  • Based on MVC architecture
  • Data binding (bi-directional)
  • Scalable
  • Can be used on standalone basis
  • Handy tools for developers
  • Better Error Handling
  • A large community of developers and supporters
  • Works across multiple platforms, including mobile.
  • Supported any modern browser and web technologies
  • Used by Google, Forbes, Microsoft, Autodesk, Apple, Adobe, Upwork, Telegram...


3. The Downsides to Angular

  • Not good for Search Engine Optimization (SEO) if use CSR (Client Side Rendering)
  • Websites that need to store a lot of data are also normally slow with Angular.
  • Difficult at first

Some infographics showing the server-client rendering options


4. What was done in this Hackathon:

  • Created simple Angular MVC SPA CSR web application with side navigation menu and responsive layout.
